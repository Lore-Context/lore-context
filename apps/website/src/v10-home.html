How it works

A simple memory loop for non-technical users.

Sign in once, connect your tools, approve what matters, and let Lore handle the memory layer in the background.

01

Connect agents

Google sign-in creates a personal vault. Lore then connects agent sessions, web AI, and later document connectors with user consent.

02

Review memory

Captured facts land in Memory Inbox. You approve, edit, reject, delete, or mark sensitive facts private before they become durable.

03

Recall with receipts

Every connected agent can recall approved memory. Evidence Ledger records source, timestamp, conflicts, and whether the memory was used.
